Gerenciador de Salas de Reuniões

API REST em Spring Boot e SPA em Angular para cadastro e gerenciamento de salas de reunião.

Visão Geral do Projeto

Este projeto oferece uma solução completa para o gerenciamento de salas de reuniões, combinando um robusto backend em Java com Spring Boot e um frontend interativo desenvolvido em Angular.

Ideal para empresas que buscam otimizar a reserva e organização de seus espaços de reunião.

Funcionalidades Principais

  • Cadastro e listagem de salas de reunião.
  • Visualização detalhada de cada sala.
  • API RESTful para integração flexível.
  • Interface de usuário moderna e responsiva com Angular.

Tecnologias Utilizadas

Backend

  • Java 11+
  • Spring Boot 2.x
  • Spring Data JPA
  • H2 Database (in-memory)
  • Lombok

Frontend

  • Angular 10
  • TypeScript
  • RxJS
  • Angular Forms & HttpClient
